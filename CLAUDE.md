# 遺伝子生成

このディレクトリは遺伝子生成のディレクトリです。
スキルを使って遺伝子生成を行います。


## ディレクトリ構造

```
{gene_type}/
  criteria.md              ← 掛け合わせ方・評価基準・遺伝子数パラメータ
  gene/
    {世代数}/
      all_genes.json       ← その世代の全遺伝子（1ファイルに一括管理）
      evaluation.md        ← 評価結果とランキング（evaluator実行後に生成）
```

gene_type: 遺伝子タイプ名（例: nazokake）
世代数: 001のように、3桁の前ゼロつきです。
